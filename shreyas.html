MediNest

AI-powered medical learning platform · FastAPI + React + Next.js + TypeScript

The most technically ambitious app in the ecosystem. The frontend is a React + TypeScript + Next.js application styled with Tailwind CSS — component-driven, fully type-safe, and deployed via GitHub to Render. The backend is a FastAPI service with PostgreSQL, containerised with Docker.

Integrates Claude AI (medical assistant), DeepL (multilingual support), PubMed API (research lookup), and a custom Three.js 3D anatomy explorer.

Micro-Frontend Gateway · Flask Proxy
👑

CrownDesk Hub

Reverse-proxy gateway + super-admin panel for the full ecosystem

The command-centre that ties all four apps together. A Flask reverse-proxy routing /jobdesk, /nexora, /medinest to their respective services under one domain — architecturally equivalent to a micro-frontend shell. Includes a unified super-admin panel, CSRF middleware, live service health checks, and a Free Beta Mode feature flag for launch gating without code changes.
