👑 CrownDesk Hub

The Ecosystem Gateway

The master hub that powers the entire ecosystem — a Flask proxy layer routing all traffic under one custom domain with a centralised super-admin panel.

Smart Reverse Proxy — Routes /jobdesk, /nexora, /medinest to internal services transparently.
Super-Admin Panel — Unified admin that manages all three sub-apps from one interface.
CSRF Protection — Ecosystem-wide CSRF middleware with per-service exemptions.
Free Beta Mode — Launch-safe posture with payment-disabled mode until full legal/tax setup.
